# Vue Setup
## Install vue globally
```
sudo npm i -g @vue/cli
```
> Graphical user interface for vue projects
```
vue ui
```
> create new project, 'babel, router', 'vue3', 'in dedicated files'
```
vue create [app-name]
```
> start server
```
npm run serve
```
> create production build
```
npm run build
```
